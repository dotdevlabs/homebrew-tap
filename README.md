# dotdevlabs Homebrew Tap

This is the official Homebrew tap for [dotdevlabs](https://github.com/dotdevlabs) CLI tools.

## Install

```sh
brew tap dotdevlabs/tap
```

### clusterctl

Cluster lifecycle management CLI tool.

```sh
brew install dotdevlabs/tap/clusterctl
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `clusterctl` | Cluster lifecycle management CLI tool |
